Stéphane Bessy: Oriented trees in O(k√k)-chromatic digraphs, a subquadratic bound for Burr's conjecture

In 1980, Burr conjectured that every directed graph with chromatic number 2k−2 contains any oriented tree of order k as a subdigraph. Burr showed that chromatic number (k−1)^2 suffices, which was improved in 2013 to k^2/2−k/2+1 by Addario-Berry et al. We give the first subquadratic bound for Burr's conjecture, by showing that every directed graph with chromatic number 8 √{2/15} * k√k+O(k) contains any oriented tree of order k. In this talk, I will mainly focus on the case of (out-)arborescences, where we obtain the improved bounds of 4/3 * k√k + O(k).
This is joint work with D. Gonçalves and A. Reinald

6. Tomasz Kociumaka (MPI-INF, Germany): Text Indexing in Compressed Space

Friday, 2026-05-22, 12:00

Slides

Massive sequential datasets in genomics and other areas have intensified the challenge of designing efficient text indexes: data structures that answer queries about a dataset modeled as a very long string (the text). A fundamental example is pattern matching, which asks whether a short string given at query time (the pattern) occurs in the text. Classic structures such as suffix arrays, and their compact variants like the FM-index, support such queries but do not scale well for highly repetitive data, including population-scale genome collections and versioned document repositories. These settings require indexes whose size is comparable to that of standard compression methods.
In this talk, I will introduce the area of compressed text indexing, discuss selected state-of-the-art results, and highlight several future directions. On a more technical level, I will present a simple compressed text index for pattern matching that combines two tools I contributed to the area: substring complexity and synchronizing sets.

5. Joy Qiping Yang (University of Sydney): Parameterized Uniformity Testing in the Huge Object Model

Friday, 2026-04-24, 12:00

We continue the study of distribution testing in the Huge Object Model (Goldreich and Ron, 2022), focusing on the flagship question of \emph{parameterized uniformity testing}: given access to data in $\{0,1\}^n$, is its underlying probability distribution uniform on a small subset of size $m$? The focus here is on both the number of data points required (sample complexity) and the number of bits from these data points one needs to observe (query complexity). While previous work fully characterized both of these in the ``tiny support'' regime $m \ll n$, very little was understood in the ``medium-to-large'' support one ($n \leq m \ll 2^n$). Our work remedies this state of affairs, establishing the existence of a sample-query complexity tradeoff and showing that (in contrast to the tiny-support regime) adaptive algorithms can significantly outperform non-adaptive ones.
This is joint work with Clément Canonne and Sayantan Sen.

4. Philip Wellnitz: The Parameterized Complexity of Counting Small Sub-Hypergraphs

Friday, 2026-04-10, 12:00

Subgraph counting is a fundamental problem whose computational complexity is well understood. Quite surprisingly, the hypergraph version of subgraph counting has been almost ignored. In this work, we address this gap by investigating the most basic sub-hypergraph counting problem: given two hypergraphs H and G, compute the number of sub-hypergraphs of G isomorphic to H. Formally, for a family C of hypergraphs, let \#Sub(C) be the restriction of the problem to H in C; the induced variant \#IndSub(C) is defined analogously.
Our main contribution is a complete classification of the fixed-parameter tractability of these problems. Assuming the Exponential Time Hypothesis, we prove that \#Sub(C) is fixed-parameter tractable if and only if C has bounded fractional co-independent edge-cover number, a novel graph parameter introduced in this work, and that \#IndSub(C) is fixed-parameter tractable if and only if C has bounded fractional edge-cover number. Both results subsume preexisting results for graphs as special cases. We also show that the fixed-parameter tractable cases of \#Sub(C) and \#IndSub(C) are unlikely to be in polynomial time, unless respectively \#P=P and Graph Isomorphism is contained in P. This shows a separation with the special case of graphs, where the fixed-parameter tractable cases are known to actually be in polynomial time.
Joint work with Marc Roth, Marco Bressan, Julian Brinkmann, and Holger Dell.

2. Ian Mertz: Computing With A Full Memory: A Primer

Wednesday, 2026-03-18, 12:00

Can memory be an asset even when full? In the catalytic computing model, a space-bounded machine is given additional access to an additional, much larger work tape; however, this tape is initialized with arbitrary data that must be reset at the end of the computation. Surprisingly, such memory can provide a large amount of additional power, even exceeding other resources such as randomness or non-determinism.
In this talk we will survey the basics of catalytic computing, from central techniques to the greatest successes of the field, as well as possible ways to incorporate this new paradigm into other areas of computer science.

1. Ararat Harutyunyan: Coloring Kneser and multi-partite digraphs

Friday, 2026-03-13, 12:00

The dichromatic number of a digraph D is the smallest k such that D can be vertex-partitioned into k acyclic subdigraphs, and the dichromatic number of an undirected graph is the maximum dichromatic number over all its orientations. We show that the dichromatic number of the Kneser graph KG(n, k) is Θ(n − 2k + 2). We then study the list version of the dichromatic number, showing that the list dichromatic number of KG(n, k) is Θ(n ln n) under certain conditions on n and k. This extends a recent result of Bulankina and Kupavskii on the list chromatic number of KG(n, k), where the same behaviour was observed. We also find that the list dichromatic number of the complete r-partite graph with m vertices in each part is Θ(r ln m), extending a result of Alon on undirected graphs.
Joint work with Gil Puig i Surroca.
